## 🚀 Featured Projects

### 🛳️ MPAIS — Maritime Port Management System
A government-grade port management platform for Latakia Port, built as a Next.js + Express.js + PostgreSQL monorepo. Features real-time AIS vessel tracking via WebSocket (aisstream.io), port fee calculation across six fee types, Pratique clearance workflows, and a full NIST-based security architecture.

`Next.js` `Express.js` `PostgreSQL` `Socket.IO` `TypeScript` `Docker`

---

### 📊 Bridge MENA — Analytics Dashboard
A full-stack analytics dashboard built with Next.js 16, NestJS, RabbitMQ, and Firebase. Features real-time data pipelines, role-based access control, and event-driven microservice communication.

`Next.js 16` `NestJS` `RabbitMQ` `Firebase` `TypeScript`

---

### 👥 FlowHR — Human Resources Platform
An HR management system with employee lifecycle management, leave tracking, and payroll modules. Built with Next.js and MongoDB, with a comprehensive JWT-based auth layer.

`Next.js` `MongoDB` `JWT` `TypeScript`
